# STM32 高级定时器基础时基（TIM1/TIM8）示例

## 📌 项目简介

本项目演示如何将 STM32F10x 高级定时器（TIM1 / TIM8）当作基础定时器使用，
实现周期性定时与中断计数功能。

---

## ⚙️ 功能特点

- 使用 TIM8 作为基础时基
- 配置 PSC + ARR 实现 1ms 定时
- 使用更新中断计数实现 1 秒周期
- 控制 LED 周期翻转

---

## ⏱️ 定时计算

系统时钟：72 MHz

PSC = 72 - 1  → 计数频率 = 1 MHz（1us）
ARR = 1000 - 1 → 1ms 更新中断

👉 每 1000 次中断 = 1 秒

---

## 🔁 工作流程

1. 初始化 TIM8
2. 开启更新中断
3. 每次中断计数 +1
4. 达到 1000 次 → LED 翻转

---

## 🧠 关键说明

- 虽然使用 TIM8，但这里只使用“基础定时功能”
- 没有使用 PWM / BDTR / 互补输出
- advancedtimer_cnt 是软件计数，不是 CNT 寄存器
